# 🎵 音樂數據分析 Dashboard

## 簡介

Spotify × Discogs × MusicBrainz 三平台的綜合分析儀表板，涵蓋 2021-2026 年度的音樂數據。

### 📊 包含內容

- **🎧 Spotify**: 28 位頂級藝術家，17.4 億粉絲，12 個流派分析
- **🎚️ Discogs**: 15 位藝術家評分數據，平均評分 4.47/5.0
- **🏷️ MusicBrainz**: 14 位藝術家標籤分析，15 個熱門標籤

### 📈 視覺化圖表

- 流派分佈（橫向條形圖）
- 年份趨勢（折線圖）
- 藝術家排名（彩色條形圖）
- 評分分佈（直方圖）
- 標籤雲（熱力圖）
- 國家分佈（地理分析）

### 🎨 特色

✅ **完全互動式** - 使用 Plotly.js  
✅ **響應式設計** - 在任何設備上完美顯示  
✅ **深色模式友善** - 梯度紫色主題  
✅ **無依賴** - 所有資料嵌入 HTML，無需外部 API  
✅ **可離線使用** - 下載檔案即可本地開啟  

### 🚀 使用方式

1. 打開 `index.html` 在瀏覽器中
2. 點擊上方 Tab 切換三個平台的數據
3. 懸停圖表查看詳細數值
4. 拖曳、縮放、下載圖表

### 📱 相容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 🔄 更新數據

要更新數據，需要：
1. 編輯原始 CSV 檔案
2. 執行 Python 數據處理腳本
3. 重新生成 `index.html`

聯繫維護者以獲取完整的數據管道。
